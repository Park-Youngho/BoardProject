# 1. 프로젝트 소개 (2025-02-10 ~)
> 백엔드 기초를 다지기 위해 게시판 프로젝트를 제작하였습니다.

# 2. 구현된 기능
+ 회원가입
+ 로그인
+ 게시판 작성
+ 로그인 검증
+ 세션기능(세션쿠키를 통해 사용자가 새로고침하여도 로그인 유지 로그아웃시 세션쿠키 삭제)
+ 로그아웃
+ 허가되지 않은 URI는 로그인 화면으로 이동(비회원)

# 3.사용 기술
+ 백엔드(프레임워크/라이브러리)
  + SpringBoot
  + Spring-MVC
  + Spring-data-jpa
  + lombok

+ 데이터베이스
  + H2 database


+ 프론트엔드
  + thymeleaf
  + HTML/CSS
  + Bootstrap
